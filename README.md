## üíª How to Run

### For C Programs:
```bash
gcc filename.c -o output
./output
```

### For C++ Programs:
```bash
g++ filename.cpp -o output
./output
```

Make sure you have GCC/G++ installed on your system. On Linux:
```bash
sudo apt install build-essential
```
